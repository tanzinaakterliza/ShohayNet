
# ShohayNet

## Disaster Help Coordination Platform

Connecting affected people, volunteers, NGOs, donors, and government agencies in one secure platform. ShohayNet provides fast emergency support, real-time updates, and efficient relief coordination during disasters across Bangladesh.

## Project Overview

ShohayNet is a web-based disaster management and humanitarian coordination platform developed to support disaster response and recovery operations. The system integrates disaster monitoring, volunteer coordination, resource management, emergency reporting, communication, aid requests, donations, and government service integration into a single platform.

## System Modules

### 1. Real-Time Disaster Updates

Provides live disaster alerts and situational updates to help users stay informed during emergencies.

#### Features
- Live disaster alerts and notifications
- Flood, cyclone, earthquake, and wildfire monitoring
- Incident details and emergency updates
- Disaster filtering and tracking

![Real-Time Disaster Updates](screenshots/disaster-updates.png)



### 2. Live Location Mapping

Displays shelters, rescue teams, relief distribution points, and risk zones on an interactive map.

#### Features
- Shelter location identification
- Rescue team tracking
- Relief distribution monitoring
- Risk zone visualization
- GPS-based disaster mapping

![Live Location Mapping](screenshots/live-location-mapping.png)



### 3. Resource Inventory Tracking

Manages relief resources and monitors their availability during disaster response operations.

#### Features
- Medical supply management
- Food and water inventory monitoring
- Shelter equipment tracking
- Resource availability updates
- Check-in and check-out system

![Resource Inventory Tracking](screenshots/resource-inventory.png)



### 4. Volunteer Sign-up & Assignment

Allows volunteers to register, receive assignments, and participate in relief operations.

#### Features
- Volunteer registration
- Skill-based assignment
- Mission tracking
- Volunteer dashboard
- Service hour monitoring

![Volunteer Sign-up & Assignment](screenshots/volunteer-signup.png)



### 5. Emergency Reporting Tool

Enables users to report emergency incidents and request immediate assistance.

#### Features
- Emergency incident reporting
- GPS location sharing
- Photo evidence upload
- Severity level classification
- Emergency contact integration

![Emergency Reporting Tool](screenshots/emergency-reporting.png)



### 6. Chat & Communication Hub

Facilitates communication between responders, volunteers, and coordinators.

#### Features
- Real-time messaging
- Emergency team coordination
- Priority message alerts
- Group communication support
- End-to-end encrypted communication

![Chat & Communication Hub](screenshots/chat-communication.png)



### 7. Aid Request Form

Allows affected individuals and families to request emergency assistance.

#### Features
- Food assistance requests
- Water support requests
- Medical aid requests
- Rescue service requests
- Urgency-based request management

![Aid Request Form](screenshots/aid-request.png)



### 8. Offline Mode Support

Provides emergency support functionality when internet connectivity is unavailable.

#### Features
- SOS request submission
- Offline emergency reporting
- Emergency data collection
- Location capture support

![Offline Mode Support](screenshots/offline-mode.png)



### 9. Data Analytics Dashboard

Provides insights into disaster response activities through visual statistics and reports.

#### Features
- Incident statistics
- Affected population tracking
- Resource deployment analytics
- Volunteer activity monitoring
- Disaster impact analysis

![Data Analytics Dashboard](screenshots/analytics-dashboard.png)



### 10. Government Service Integration

Supports coordination with government agencies and humanitarian organizations.

#### Features
- Disaster management authority coordination
- Humanitarian cluster support
- Improved transparency
- Faster emergency response
- Better resource allocation

![Government Service Integration](screenshots/government-integration.png)



### 11. Donation Management

Allows individuals and organizations to financially support disaster relief efforts.

#### Features
- Online donation submission
- Multiple payment methods
- Donor information management
- Optional donor messages
- Secure contribution process

#### Donation Information
- Full Name
- Mobile Number
- Email Address
- Donation Amount (BDT)
- Payment Method
- Optional Message

![Donate Now](screenshots/donate-now.png)
